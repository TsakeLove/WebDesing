Лабораторна робота #1

Таблиці

Grid модуль для CSS был разработан рабочей группой CSS для того, чтобы сделать создание шаблонов в CSS максимально удобным. Створення таблиць було реалізовано за допомогою grid. Grid модуль для CSS був розорблений робочею групою для того, щоб полегшити створення CSS шаблонів. Саме цей спосіб дозволяє створювати адаптивні сторінки й таблиці.

.new { display: grid; grid-template-areas: "header-new" "body-new"; grid-template-rows: 65px 1fr 100px; grid-template-columns: 120px 200px 100px; }
